# Cau
## 智能排课系统 — 自助选课方案生成器 使用帮助

### 系统简介

智能排课系统是一个纯前端的单页 Web 应用，无需安装任何软件。打开 HTML 文件即可使用。系统内置了一份「生成模板」默认课表供参考，上传你的课程 Excel 后，系统会采用多策略分阶段算法自动生成 20 套选课方案，并支持优选教师机制，帮你快速找到最合适的课表组合。

---

### 快速上手

**第一步：打开页面**

直接双击 `预选排课助手.html` 文件，在浏览器中打开。页面会展示一份内置的「生成模板」课表作为参考。

**第二步：准备 Excel 文件**

从学校教务系统导出你的课程数据，保存为 `.xlsx` 格式。Excel 中需要包含以下列（列名必须完全一致，不含多余空格）：

| 列名 | 说明 | 是否必填 |
|------|------|----------|
| 课程名称 | 课程全称，如"高等数学" | 必填 |
| 教师姓名 | 授课教师名称 | 必填 |
| 上课时间 | 如 `周一第1,2节` 或 `周四第3,4节{第5-16周}(全部)[教室]` | 必填 |
| 上课周次 | 如 `1-16`（连续）或 `3,5,7`（不连续） | 必填 |
| 限选人数 | 课程容量上限（数字） | 必填 |
| 班级人数 | 当前已选人数（数字） | 必填 |
| 上课地点 | 教室或地点 | 选填 |
| 学分 | 课程学分 | 选填 |

> 系统支持教务系统导出的原始时间格式，`{第X-X周}` 和 `[教室]` 等后缀会被自动识别和忽略，无需手动修改。

**第三步：上传文件**

点击页面左侧的上传区域，或将 Excel 文件直接拖拽到上传区域。系统会自动解析文件内容并生成方案。

**第四步：标记优选教师（可选）**

上传成功后，左侧会出现「优选教师」面板。点击老师姓名可以标记为优选（再次点击取消）。标记后，系统会自动重新生成方案，优先推荐包含优选教师课程的组合。

**第五步：浏览和选择方案**

左侧方案列表会显示所有生成的方案，点击任意方案可在右侧查看详细的课程清单和周课表。

### Excel 工作表（Sheet）组织方式

Excel 文件可以包含多个工作表，系统按以下规则处理：

**已选课程（固定课表）**

名为 `已选` 或 `已选课程` 的工作表会被视为已选课程。这些课程在所有方案中都会出现，不会被重复选择。把你已经确定要上的课放在这里。

**课程模块**

其他每个工作表被视为一个独立的课程模块，工作表名即为模块名。每套方案会从每个模块中各选一门课进行组合。例如：

- `通识课` 工作表 → 通识课模块，每套方案从中选 1 门
- `体育课` 工作表 → 体育课模块，每套方案从中选 1 门
- `已选课程` 工作表 → 固定课表，所有方案都包含

同一模块内的课程是互斥的（只能选一门），不同模块的课程可以自由组合。

**自动跳过**

系统会自动跳过所有以 `microsoft.com:` 开头的内部工作表，无需手动处理。

### 校区筛选

如果你的课程数据中同时包含东校区和西校区的课程，上传前请先通过 Excel 的「校区」列筛选，删除非你所在校区的课程行，只保留你所在校区的课程，避免生成无法前往上课的方案。

---

### 方案生成策略

系统每次生成 20 套方案，分四个阶段：

**优选推荐** — 如果你标记了优选教师，系统会优先组合包含这些教师课程的方案，排在列表最前面。

**容量优先（约 6 套）** — 优先选取剩余名额多的课程，选课成功率高。

**时间均衡（约 6 套）** — 兼顾容量与上课时间分布，按「少早八」「少晚课」「单日最均衡」等策略排序。

**随机探索（约 8 套）** — 从剩余合法组合中随机选取，提供多样化选择。

### 界面说明

**左侧面板**

- 上传区：点击或拖拽上传 Excel 文件，显示解析状态
- 优选教师面板：上传后显示教师名单，点击标记优选
- 方案列表：显示所有方案概览，点击切换查看详情

**右侧面板**

- 课程详情表：显示当前方案的课程清单，包含模块、课程名称、学分、教师、时间、地点、剩余名额等信息
- 周课表：以周一到周日、六大节的网格形式直观展示课程安排

**颜色标识**

不同模块的课程在课表中以不同颜色区分，优选教师的课程以黄色高亮显示。剩余名额用绿色（充足）、橙色（一般）、红色（紧张）标识。

### 常见问题

**打开页面后看到什么？**
页面内置了一份「生成模板」课表作为参考，打开即可看到。上传你的 Excel 后会自动切换为智能生成的方案。

**上传后提示"未找到有效课程数据"？**
检查 Excel 列名是否包含六列必填项且完全匹配：课程名称、教师姓名、上课时间、上课周次、限选人数、班级人数。注意列名不含多余空格。

**上课时间格式很复杂，需要手动修改吗？**
不需要。系统支持教务系统导出的原始格式，如 `周四第3,4节{第5-16周}(全部)[工院楼-267]`，系统会自动解析。

**优选教师不生效？**
标记优选教师后方案会自动重新生成。含有优选教师课程的方案会排在列表前面，并以紫色标签「优选推荐」标识。

**什么是"早八"和"晚课"？**
早八 = 第 1-2 节（08:00-09:50），晚课 = 第 9 节及以后（19:00-）。

**可以自己定义模块名称吗？**
可以。工作表的名称就是模块名称，你可以将工作表改名为"通识课""专业课"等，系统会按这些名称生成模块标签。

### 技术说明

本系统为纯前端应用，所有数据处理均在浏览器本地完成，不会上传任何数据到服务器。依赖 SheetJS（xlsx）库解析 Excel 文件，通过 CDN 加载。支持主流现代浏览器（Chrome、Edge、Firefox、Safari）。
